# WanderGuide

WanderGuide is a modern travel planning application designed to help users discover destinations, plan itineraries, and manage their trips.

## Features

-   **Destination Discovery**: Explore curated destinations with "National" and "International" categories.
-   **Itinerary Planning**: View detailed 3-day itineraries for various locations.
-   **User Dashboard**: personalized dashboard with upcoming trips and request status.
-   **Custom Requests**: Request personalized itineraries from travel experts.
-   **User Settings**: Manage profile, currency, and travel preferences.
-   **Authentication**: Secure login flow for users.

## Tech Stack

-   **Frontend**: React, TypeScript, Vite
-   **Styling**: Tailwind CSS, Lucide React (Icons), Framer Motion (Animations)
-   **State/Storage**: LocalStorage-based mock backend (store.ts)
-   **Routing**: React Router DOM

## Getting Started

1.  Clone the repository.
2.  Run `npm install` to install dependencies.
3.  Run `npm run dev` to start the development server.
